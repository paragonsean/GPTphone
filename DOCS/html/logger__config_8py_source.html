94 This method is a decorator that times the execution of a given function. It wraps the function with a timer and logs the elapsed time using the logger module. The timer uses the `perf_counter()` method from the `time` module to measure the elapsed time.
95
96 Usage:
97 @timer
98 def my_function():
99 # code to be timed
100
101 Example:
102 @timer
103 def fibonacci(n):
104 if n <= 1:
105 return n
106 else:
107 return fibonacci(n-1) + fibonacci(n-2)
108
109 Note: The logger module must be properly configured before using the `timer` decorator. It's recommended to initialize the logger in the calling module before using this decorator.
110 """
111 @functools.wraps(func)

113 start_time = time.perf_counter() # Start the timer
